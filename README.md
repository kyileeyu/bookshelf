# Multi Step Form


## 기술스택
	- typescript
	- next
		- page router
	- react
	- react-query
	- react-hook-form
	- emotion
	- (필요할때만) jotai



애플리케이션은 5단계로 구성

- 1단계 - 도서 기본 정보, 독서 상태, 독서할 시작 및 종료일
  - 독서 상태는 읽고 싶은 책 / 읽는 중 / 읽음 / 보류 중
- 2단계 - 도서 추천 여부, 별점
- 별점은 0~5, 0.5점 스케일
- 3단계 - 독후감
- 4단계 - 인용구
- 5단계 - 공개 여부

### 세부 요구사항 

유효성 검증

- [x] 독서 상태 & 독서 기간
- [x] 읽고 싶은 책 은 독서 기간이 입력되면 안 된다.
- [x] 읽는 중 이면 독서 시작일만 입력돼야 한다. / 독서 종료일은 입력되면 안 된다.
- [x] 읽음 이면 독서 시작일/종료일이 입력돼야 한다.
- [x] 보류 중 이면 독서 시작일만 입력돼야 한다. / 독서 종료일은 입력되면 안 된다.
- 독서 기간
- [x] 독서 시작일은 독서 종료일보다 이후면 안 된다.
- [x] 독서 시작일은 도서 출판일 이후여야 한다.
- [x] 별점 & 독후감
- [x] 별점이 1점 또는 5점의 경우, 의견을 뒷받침 하기 위해 최소 100자 이상을 작성해야 한다. 2~4 점 일때는 독후감 필드는 입력안해도 된다.
- [ ] 인용구 페이지 번호는 도서 전체 페이지 수보다 작아야 한다.
